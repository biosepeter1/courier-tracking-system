🔍 CourierTrack Environment Check

This page helps verify your environment variables are set correctly.

Expected Configuration:

VITE_API_URL: https://courier-tracking-system-nkdz.onrender.com/api

VITE_SOCKET_URL: https://courier-tracking-system-nkdz.onrender.com

Instructions:

  1. Go to Vercel Dashboard → Your Project → Settings
  2. Click "Environment Variables"
  3. Add both variables with "Production" selected
  4. Redeploy from the Deployments tab
